# ATEL MCP OpenClaw Plugin

This plugin registers a single OpenClaw tool, `atel_mcp`, that bridges OpenClaw to the ATEL Remote MCP server over OAuth.

Supported actions:

- `list`: list remote ATEL MCP tools
- `call`: call a remote ATEL MCP tool with JSON arguments

Default target:

- `https://43-160-230-129.sslip.io`
